# 📚 Estudos sobre LLMs (Large Language Models)

Este diretório é dedicado ao meu aprendizado contínuo sobre **LLMs (Large Language Models)**, explorando desde conceitos fundamentais até aplicações práticas no desenvolvimento de software.

O objetivo aqui não é apenas “usar” LLMs, mas **entender como funcionam**, quais são suas limitações, como avaliá-los criticamente e como integrá-los de forma responsável em sistemas reais.

---

## 🎯 Objetivos

- Entender os fundamentos de LLMs e NLP
- Aprender como modelos como GPT, Claude, LLaMA e outros funcionam
- Explorar técnicas de **prompt engineering**
- Estudar embeddings, contexto, tokens e custo computacional
- Criar pequenos experimentos e provas de conceito
- Avaliar riscos, limitações e boas práticas no uso de IA

---

## 🧠 Conceitos abordados

Alguns dos temas que pretendo estudar e documentar aqui:

- O que são LLMs e como são treinados
- Tokens, contexto e janela de atenção
- Prompt Engineering (zero-shot, few-shot, chain-of-thought, etc.)
- Embeddings e busca semântica
- Fine-tuning vs RAG (Retrieval-Augmented Generation)
- Avaliação de respostas e alucinações
- Ética, vieses e limitações dos modelos
- Casos de uso práticos em produtos e sistemas
